# Changelog

All notable changes to DeepRhapsody will be documented in this file.

## [0.1.0] - Unreleased

### Added
- Debug session scripts for 8 languages: Python, C/C++, C#, Rust, Java, Go, Node.js, Ruby
- GitHub Copilot CLI integration (agent + skill)
- MCP server for Claude Desktop / Cursor / any MCP-compatible client
- OpenAI function calling adapter + pre-built schemas
- LangChain / AutoGen / CrewAI tools
- Universal system prompt for shell-based agents
- Tutorials for all supported AI platforms
- C# example project with 5 intentional bugs
- Python example project (sample_buggy_grades.py)
- CI workflow for syntax validation
